# Implemented Backup Solutions

- **Proxmox Host Backup**  
  - Block-level backup of the Proxmox system using **Clonezilla**  
  - Automation with **preseed** configuration

- **Virtual Environment Backup**  
  - Backing up VMs and LXCs to a virtualized **Proxmox Backup Server**

- **Client-side Backups**  
  - Backing up my Windows laptop using **Veeam Backup & Replication Community Edition** to an SMB share  
  - **Personal files backup and synchronization**  
  - **Nextcloud**: self-hosted file sharing between laptop and phone  
  - **Phone**: one-way photo sync to homelab using **FolderSync**  
  - **Laptop**: file synchronization to homelab using **FreeFileSync**, later replaced with **Restic**
