# Macrium Reflect

- I use **Macrium Reflect** to create a **full image backup of my dual-boot laptop (Windows 10 and Ubuntu 22.04)**.  
- Especially reliable for machines with multiple operating systems, as it backs up the entire system at the **disk image level** (including boot partitions).
